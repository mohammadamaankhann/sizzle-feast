**Sizzle Feast** is a modern, high-performance static web application designed for premium food brands. It features a sleek dark aesthetic, ultra-responsive design, and a seamless user experience optimized for conversion.

---

## 🚀 Key Features

- **💎 Premium Aesthetic**: Modern dark theme with high-contrast red accents (KFC inspired).
- **📱 Fully Responsive**: Optimized for all devices, from mobile phones to high-resolution desktops.
- **⚡ Clean URLs**: User-friendly URLs without `.html` extensions (e.g., `/about` instead of `/about.html`).
- **🔍 Dynamic Navigation**: Smart category dropdowns with hover support for instant access to menus.
- **📄 Professional Legal Pages**: Redesigned Privacy Policy and terms sections with clean layouts.
- **🍔 Interactive Menus**: Dedicated pages for Zinger Burgers, Crispy Chicken, and Wraps & Rolls.

---

## 🛠️ Tech Stack

- **Frontend**: Standard HTML5 & Vanilla CSS3
- **Framework**: Bootstrap v5.1 (Grid & Components)
- **Icons**: Bootstrap Icons & Font Awesome
- **Typography**: Inter Sans-serif for a modern, professional look
- **Infrastructure**: Apache URL Rewriting via `.htaccess`

---

## 📂 Project Structure

```text
├── images/             # Optimized brand assets and product images

```

## 📥 Getting Started

### Prerequisites
- Any modern web server (Apache recommended for Clean URLs support)
- Alternatively, you can open `index.html` directly in any browser.

### Installation
1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/sizzle-feast.git
    ```
2.  **Navigate to the directory**:
    ```bash
    cd sizzle-feast
    ```
3.  **Deployment**:
    - Upload all files to your server's root directory.
    - Ensure `.htaccess` is present to enable Clean URLs.
